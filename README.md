# Flashtract

# To run this project follow these steps:
  First:
    From terminal: 
      go to 'build/libs' folder.
    
  Second:
    Execute the .jar in this folder: 
      'java -jar flashtract-0.0.1-SNAPSHOT.jar'

# Note: 
   Please notice this project runs on 8081 port.

# Swagger
